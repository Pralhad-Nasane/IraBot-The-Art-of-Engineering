What is coding or programming?

Coding is the language that a computer can understand. Like if we wanted to give an order in a restaurant, we speak a specific language so that the next person would understand us, and then we will get our order. In the same way, if we want our machine to work as per our order, we need the language which the machine would understand and that language is called coding or programming.

what actually this course contains?

In this course, students would learn the integration of hardware with software by using coding.
Students will create their own machines and robots with coding because the whole course is based on project based learning.
Kids’/students’ imagination power is bigger than ours, so if they will get the correct guidance for learning, there is no limitation to creating the software and hardware machines on their own.
